# Purple Islands 3JS

A captivating 3D web experience featuring procedurally generated crystalline islands rendered with Three.js and custom GLSL shaders. Explore a surreal landscape of purple-hued terrains that dynamically shift and shimmer as you navigate.

## ✨ Features

- **Hexagonal Terrain Grid**: Procedurally generated islands arranged in a hexagonal pattern
- **Custom Shaders**: Advanced GLSL vertex and fragment shaders for realistic lighting, reflections, and noise effects
- **Interactive Controls**: Orbit controls for zooming and panning through the 3D scene
- **Dynamic Animation**: Time-based noise deformation creates living, breathing landscapes
- **Responsive Design**: Adapts to different screen sizes and resolutions

## 🚀 Technologies Used

- **Three.js**: Powerful 3D JavaScript library for WebGL rendering
- **GLSL Shaders**: Custom vertex and fragment shaders for advanced visual effects
- **HTML5 Canvas**: Hardware-accelerated graphics rendering
- **JavaScript ES6**: Modern JavaScript with classes and advanced features

## 🖥️ How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Enjoy the immersive 3D experience!

No additional dependencies or build steps required - it's pure client-side magic!
